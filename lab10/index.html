Lab 10 - JavaScript for the Web

Challenge

This lab used jQuery to add interaction to a webpage. The goal was to create a button that shows new content with each click. It also helped practice using event listeners.

Problems

There were issues with file paths and making sure the script loaded correctly. Getting the button to reset the content also took some trial and error.

Results

The button shows one line of a poem at a time, and resets when finished. The jQuery works as expected and the layout looks clean with custom styling.
