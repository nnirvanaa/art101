Lab 10 - jQuery Interaction

Challenge

The goal of this lab is to use jQuery to create a simple interactive webpage. The main task is to take input from a user and display a response using JavaScript. This builds a better understanding of how jQuery can interact with HTML elements dynamically.

Problems

Some difficulty came from making sure the jQuery library was linked correctly and that JavaScript only ran after the page loaded. There were also small issues with making sure the output displayed in the right place. These were fixed by checking file paths and using proper selectors in jQuery.

Results

The page takes user input and displays a message using jQuery. Typing a name and clicking the button shows the name sorted in alphabetical order. This confirms the button, input field, and script are all connected and working as expected.
