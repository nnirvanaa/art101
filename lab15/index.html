Lab 15 - Experimenting with jQuery and AJAX

Challenge

This lab explored how to use jQuery in combination with AJAX and DOM manipulation. Instead of calling a public API, this version randomly selects and displays a local image from a predefined list using jQuery.

Problems

The challenge was learning how to use jQuery to select elements, handle button clicks, and dynamically change an image on the page. It also involved setting up the file structure properly so local images loaded correctly.

Results

The result is an interactive page where the user can click a button and see one of four random dog images displayed from a local folder using JavaScript and jQuery.
