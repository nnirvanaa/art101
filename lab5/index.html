Lab 5: JavaScript Output

Challenge

In this lab, we were tasked with creating a JavaScript file that uses variables to store vehicle data and then outputs the results using document.writeln() into the browser. The challenge helped us understand how data can be structured in code and printed to the page.

Problems

The main issue I encountered was making sure the path to the JavaScript file was correctly written in the HTML. Once I fixed the path, the output displayed as expected. I also had to make sure the CSS and images were in the correct folders for everything to show up properly.
