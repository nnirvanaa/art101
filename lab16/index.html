Lab 16: JSON and APIs

Challenge

In this lab, we worked with JSON from an external API. We used jQuery's AJAX method to retrieve data from the XKCD API and display it dynamically on our webpage.

Problems

The biggest challenge was understanding how to extract the correct parts of the JSON object and insert them into the HTML. We also had to make sure the AJAX request only ran when the user clicked the button.
